# SPT-Headless-Startup-Script
This is a simple script to allow sipmle start up of a SPT server with headless support to boot automaticly on a windows pc at boot


1. install files from SPT and Fika to your C drive if installing to another drive letter edit the bat file to the correct drive letter.

2. download and move script to SPT\SPT folder where the SPT.Server.exe is.

3. then click windows key + R to launch run then type shell:startup

4. add a shortcut of the script to this folder drag then hold alt over the new window then unclick (stop draging)
